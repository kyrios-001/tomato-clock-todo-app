# 豪华版番茄钟学习助手

##  项目简介
这是一款基于 Python Tkinter 开发的桌面级番茄钟学习工具，集成了专注计时、任务管理与学习数据统计三大核心功能，旨在帮助用户通过番茄工作法高效管理学习时间，同时清晰追踪任务进度与学习记录。

##  核心功能
### 1. 番茄钟专注计时
- 支持自定义专注/休息时长（1-120分钟）
- 专注结束自动提醒，休息结束自动切换下一轮专注
- 窗口置顶功能，防止被其他应用打断
- 浅色/深色双主题切换，适配不同使用场景

### 2. 待办任务管理
- 支持任务添加、修改、删除与拖拽排序
- 高/中/低三级优先级标记，颜色区分一目了然
- 任务截止日期自动校验，超时任务自动变浅提醒
- 双击任务快速标记完成，已完成任务自动添加文字划线

### 3. 学习数据统计
- 实时展示今日专注时长、本周总时长与番茄完成数
- 近7天学习时长柱状图可视化，清晰呈现学习趋势
- 数据自动本地保存，重启程序不丢失记录

##  技术栈
- 语言：Python 3.x
- 图形界面：Tkinter
- 数据存储：JSON 文件本地持久化
